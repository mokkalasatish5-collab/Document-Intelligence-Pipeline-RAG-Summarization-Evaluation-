Streamlit Document QA & Text Summarization App

This Streamlit application allows users to perform Document Question-Answering (QA) and Text Summarization in English or German. Users can upload documents, extract text (PDF, images, audio, HTML, YouTube), build a vector database, and interact with an intelligent chatbot—all through a simple UI.

How to Use the Application
1. Select Language

Choose English or German from the sidebar.

2. Choose Your Task

Select either:

Document QA

Text Summarization

Document QA Features
1. Upload Documents

Upload PDFs, DOCX, Markdown, or plain text files from the sidebar’s Upload Files section.

2. OCR for Images

Upload image files (PNG, JPG, JPEG) for text extraction using OCR.

3. Audio Transcription

Upload MP3 or WAV audio files to automatically convert speech to text.

4. Import HTML Content

Paste a website URL to fetch and process HTML text.

5. YouTube Video Transcription

Enter a YouTube URL to extract and transcribe the video’s audio.

6. Create a Vector Database

Generate a vector database from uploaded documents for efficient retrieval.

7. Remove All Files

Clear all stored files through the Remove Files section.

Chat with the Document-Aware Chatbot

Ask questions and receive answers based on the uploaded content.

Text Summarization

Enter or paste text into the input box, then click Summarize to generate a concise summary.
